# PicPeek

Compare up to 9 images side by side — paste a URL or upload files. View dimensions, format, size, and optional advanced metadata (EXIF, DPI, bit depth, etc.).

## Quick start

```sh
git clone https://github.com/YeLwinOo-Steve/pic-peek.git
cd pic-peek
npm i
npm run dev
```

## Tech

- **React** + **TypeScript** + **Vite**
- **Tailwind** + **shadcn/ui**
- **framer-motion** (layout animations) · **exifr** (image metadata)

## Scripts

| Command      | Description        |
| ------------ | ------------------- |
| `npm run dev` | Start dev server   |
| `npm run build` | Production build |
| `npm run preview` | Preview build   |
